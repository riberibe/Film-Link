## 1. 事業概要

**FilmLink** は、既存事業者に「映像制作」をオプションとして組み込んでもらい、レベニューシェアで収益を得るB2B2C型の映像事業。

| 項目 | 内容 |
|------|------|
| 事業名 | FilmLink |
| カテゴリ | B2B2C型 映像オプション提供事業 |
| ターゲット（提携先） | 既存顧客との接点を持つ事業者（介護施設・工務店・ジム等） |
| ターゲット（エンドユーザー） | 提携先の顧客・入居者・施主など |
| ポジショニング | 「映像制作を単体販売しない」——既存サービスの付加価値として組み込む |
| 競合との差別化 | 直接営業ゼロモデル。提携先のブランド信頼を借りて申込ハードルを下げる |
| 現在フェーズ | Phase 0（基盤構築） |

## 2. 代表の想い

子供の頃の運動会のビデオ、家族で出かけた公園でのホームビデオ。大人になってから見返したとき、当時の空気や感情が鮮明に蘇り、胸が熱くなった経験は誰にでもある。

人はそれぞれ、固有のストーリーを歩んでいる。その瞬間をフィルムとして残し、いつでもあの日の感動に帰れるようにすること——それが映像の本質的な価値。

FilmLinkが組みたいのは、お客様の体験に本気で向き合っている事業者。単なる収益ツールとしてではなく、**貴社が生み出す「最高の笑顔」を一生の記録として残すこと**を、一緒に実現するための事業として立ち上げた。

---

## 3. ビジネスモデル（フライホイール）

```
提携先開拓
    ↓
オプション導入合意（レベニューシェア or 固定制作費）
    ↓
提携先が利用者に案内（FilmLinkは自社営業不要）
    ↓
申込 → 撮影・制作・納品
    ↓
レベニューシェア分配
    ↓
実績・口コミ蓄積
    ↓
新提携先開拓（ループ）
```

**本質的な強み**: 提携先が増えるほど、自分たちが営業しなくても案件が自動発生する構造をつくる。

## 5. ターゲット業界スクリーニング

評価軸：感情的瞬間 / 先方インセンティブ / 意思決定者距離 / 競合の少なさ（各5点満点）

| 順位 | 業界 | 合計点 | 備考 |
|------|------|--------|------|
| 🥇 1位 | 介護施設・老人ホーム | 19点 | **Phase 0の最優先ターゲット** |
| 🥈 2位 | 工務店・ハウスメーカー | 18点 | Phase 1以降に展開 |
| 🥉 3位 | パーソナルジム | 16点 | Phase 1以降に展開 |
| 3位 | ペットサロン | 16点 | Phase 1以降に展開 |
| 3位 | 葬儀社 | 16点 | Phase 1以降に展開 |

### 介護施設が最優先な理由

- 「離れた家族に施設での様子を届けたい」ニーズが明確で普遍的
- 施設側が「家族満足度・差別化・追加収益」を同時に得られる
- 映像オプションを提供している施設はほぼゼロ（競合空白地帯）
- 有料老人ホーム入居者は一次購買力が高い（月額20万円以上層を優先）

## 6. 映像の設計思想

- **価値の方向は「入居者 → 家族」一択**
- 入居者が家族に「言葉と記憶を贈る映像」を残す
- 買うのは家族側（子ども・孫）または入居者本人

### インタビューの柱（撮影時の質問設計）

1. 子どもをどう育ててきたか、何を誇りに思っているか
2. 今まで何をしてきて、最後に何を届けたいのか
3. 孫に届けたい思いは何か
