# FilmLink — 提携先別オプション設計詳細（2026-04-21）

---

## 1位：介護施設・老人ホーム

### なぜここが最強か
- 「離れて暮らす家族が親の様子を知りたい」という需要は年中・永続的に存在する
- 施設側は映像オプションを持つだけで「他施設との差別化」「家族満足度向上」「追加収益」の三拍子が揃う
- 競合がほぼゼロ。映像制作会社が介護施設に営業している事例は極めて少ない

---

### オプション一覧

#### A. 日常記録ムービー（スタンダードプラン）

| 項目 | 内容 |
|------|------|
| 内容 | 施設での日常の様子（食事・レクリエーション・散歩・趣味・スタッフとの交流）を撮影・編集 |
| 撮影時間 | 施設内60〜90分 |
| 納品尺 | 4〜6分 |
| 納品形式 | URLリンクまたはデータ渡し |
| 販売価格 | 40,000円 |
| 施設取り分 | 8,000円（20%） |
| 制作側取り分 | 32,000円（80%） |
| 提供タイミング | 入居後3ヶ月以降、年1〜2回の定期購入を促す |

**刺さる理由（エンドユーザー心理）**
遠方に住む子ども・孫が「親がどんな顔で、どんな日常を送っているか」を知りたいという需要は非常に強い。
面会に頻繁に来られない罪悪感の解消にもなる。「映像を見て安心した」という体験が次の申し込みに直結する。

**施設側へのメリット**
- 家族が「この施設に預けて良かった」という実感を持ちやすくなる
- 家族からの苦情・不満が減り、施設スタッフへの負担軽減にも繋がる
- 他施設との比較で「ここは映像で様子を届けてくれる」という強い差別化になる

---

#### B. メッセージムービー（誕生日・ギフトプラン）

| 項目 | 内容 |
|------|------|
| 内容 | 入居者から家族へ向けたメッセージを映像化。施設内の好きな場所で撮影。スタッフがさりげなくサポート。 |
| 撮影時間 | 30〜45分 |
| 納品尺 | 2〜3分 |
| 販売価格 | 25,000円 |
| 施設取り分 | 5,000円（20%） |
| 制作側取り分 | 20,000円（80%） |
| 提供タイミング | 誕生日・母の日・父の日・お盆・年末（ギフト需要が高い時期） |

**刺さる理由（エンドユーザー心理）**
「親から自分への言葉を映像で受け取る」体験は他に代替がない。
施設に預けている後ろめたさがある家族ほど、こうしたギフトを求める傾向が強い。
「もっと早くやっておけばよかった」と後悔する前に申し込む動機になる。

**施設側へのメリット**
- 入居者との関係が映像を通じて深まる
- 「記念日に映像を届ける施設」として口コミが広がりやすい
- ギフト需要があるためリピート申し込みが生まれやすい

---

#### C. ライフストーリームービー（プレミアムプラン）

| 項目 | 内容 |
|------|------|
| 内容 | インタビュー形式で人生を振り返る映像。生い立ち・仕事・家族・思い出の写真も組み込む。 |
| 撮影時間 | 2〜3時間（複数回に分割可） |
| 納品尺 | 15〜20分（フルver）+ 3分（ショートver）の2本納品 |
| 販売価格 | 100,000円 |
| 施設取り分 | 20,000円（20%） |
| 制作側取り分 | 80,000円（80%） |
| 提供タイミング | 入居時・米寿（88歳）・卒寿（90歳）・白寿（99歳）などの長寿祝い |

**刺さる理由（エンドユーザー心理）**
「おじいちゃん・おばあちゃんの人生を映像として残したい」という家族の願いを叶える唯一の手段。
亡くなった後で後悔する人が非常に多い領域。「まだ元気なうちに」という動機が生まれやすい。
高額でも「一生に一度」「かけがえのないもの」として納得して払える価格帯。

**施設側へのメリット**
- 高単価オプションのため施設の収益インパクトが大きい
- 「人生を大切にする施設」という強烈なブランドイメージになる
- 入居者・家族との絆が深まり、長期入居・口コミ紹介につながる

## 2位：工務店・ハウスメーカー

### なぜここが最強か
- 完成引き渡し日は「施主にとって人生最大級の感情的瞬間」。この日を映像で残したいという需要は自明
- 中小工務店はオーナー社長に直接話でき、意思決定が速い（担当者→稟議→本社の壁がない）
- 工務店側にとって「他社との差別化」「施主のSNSによる口コミ拡散」という強いインセンティブがある
- 「完成映像オプション」を標準提供している工務店はまだごく少数（競合空白地帯）

---

### オプション一覧

#### A. 完成引き渡し記念ムービー（スタンダードプラン）

| 項目 | 内容 |
|------|------|
| 内容 | 完成した家の外観・内装・施主家族の笑顔・鍵の受け渡しシーンを撮影・編集 |
| 撮影時間 | 引き渡し当日1〜2時間 |
| 納品尺 | 4〜6分 |
| 納品形式 | URLリンクまたはデータ渡し |
| 販売価格 | 50,000円 |
| 工務店取り分 | 10,000円（20%） |
| 制作側取り分 | 40,000円（80%） |
| 提供タイミング | 引き渡し当日（見積もり段階でオプション提示） |

**刺さる理由（エンドユーザー心理）**
数千万円をかけた家の「完成した瞬間」を映像に残したいという欲求は非常に自然。
写真だけでは伝わらない「家族の表情」「家の雰囲気」「感動の瞬間」が映像では伝わる。
子どもが成長した後に見返したり、親戚に送ったりする使い道も明確。

**工務店側へのメリット**
- 施主がSNSで映像をシェアすることで、工務店の施工実績が無料で拡散される
- 他社と比較された際「映像オプションがある」というポイントで選ばれやすくなる
- 施主満足度が向上し、アフターフォロー時の関係性も良好になる
- 紹介営業（施主→知人への紹介）が生まれやすくなる

---

#### B. 建設ドキュメンタリームービー（プレミアムプラン）

| 項目 | 内容 |
|------|------|
| 内容 | 着工〜完成まで建設過程を節目ごとに撮影・記録（基礎・上棟・外装・内装完成・引き渡し）。全工程を1本のストーリーに編集。 |
| 撮影回数 | 4〜5回（各1〜2時間） |
| 納品尺 | 8〜12分 |
| 販売価格 | 110,000円 |
| 工務店取り分 | 22,000円（20%） |
| 制作側取り分 | 88,000円（80%） |
| 提供タイミング | 着工前に提案（工事中から撮影するため事前予約必須） |

**刺さる理由（エンドユーザー心理）**
「自分の家がどうやって建ったか」を映像で見られるのは唯一無二の体験。
上棟の瞬間・構造が見える段階・壁が貼られていく過程など、完成後には絶対に見られないものを記録できる。
家族の思い出として何十年も見返せるコンテンツになる。

**工務店側へのメリット**
- 職人の技術・丁寧な施工プロセスを可視化できる最強のPR素材になる
- 施主の許諾を得た上で工務店HPや展示会でも使用可能（二次活用できる）
- 高単価のため工務店の取り分も大きい（1件22,000円）

---

#### C. 施工事例PR映像（工務店B側直接費用プラン）

| 項目 | 内容 |
|------|------|
| 内容 | 工務店がSNS・HP・展示会で使える施工事例ショート動画。施主の許諾を取った上で工務店がPRに使用。 |
| 納品尺 | 30秒〜1分（縦型・横型各1本） |
| 販売価格 | 40,000円（工務店が直接費用負担） |
| 工務店取り分 | なし（工務店が直接購入するプラン） |
| 制作側取り分 | 40,000円（100%） |
| 提供タイミング | 完成引き渡し時に合わせて撮影 |

**刺さる理由（工務店側の課題解決）**
工務店は「良い家を建てても伝えるコンテンツがない」という悩みを持っている会社が多い。
自社で動画制作する人材・ノウハウ・機材がなく、映像制作会社に頼むと1本15〜30万円かかる。
FilmLinkなら4万円で施工事例映像が作れるため、コスパが圧倒的に高い。

## 3位：パーソナルジム

### なぜここが最強か
- 体型変化・目標達成という「視覚的に最もわかりやすい成果」が映像と相性抜群
- 独立オーナー運営のジムが多く、意思決定者に直接アクセスでき稟議不要
- SNS投稿文化との相性がよく、映像を顧客がシェアすることでジムの新規集客に直結
- 「顧客の記録映像」としての映像オプションは競合ほぼゼロ

---

### オプション一覧

#### A. トランスフォーメーションムービー（スタンダードプラン）

| 項目 | 内容 |
|------|------|
| 内容 | コース終了時にビフォーアフターの体型変化＋本人の感想・取り組みを映像化 |
| 撮影時間 | 1〜1.5時間 |
| 納品尺 | 2〜4分 |
| 販売価格 | 35,000円 |
| ジム取り分 | 7,000円（20%） |
| 制作側取り分 | 28,000円（80%） |
| 提供タイミング | 3ヶ月・6ヶ月コース終了時 |

**刺さる理由（エンドユーザー心理）**
体型変化は「写真でも伝わるが、映像だと動き・表情・言葉が加わり感動が倍増する」。
がんばった3〜6ヶ月を映像として残すことで自己肯定感が高まる。
SNSでシェアしたいという意欲も生まれやすく、ジムの口コミに直結する。

**ジム側へのメリット**
- 卒業生がSNSでシェアすることで、無料の広告になる
- 「映像で成果を証明してくれるジム」として入会希望者への説得材料になる
- 高い顧客満足度が口コミ紹介を生む

---

#### B. 卒業記念ムービー（感謝・次の目標プラン）

| 項目 | 内容 |
|------|------|
| 内容 | コース卒業時にジムへの感謝・出会い・次の目標を語る映像 |
| 撮影時間 | 45分〜1時間 |
| 納品尺 | 2〜3分 |
| 販売価格 | 25,000円 |
| ジム取り分 | 5,000円（20%） |
| 制作側取り分 | 20,000円（80%） |
| 提供タイミング | コース卒業・目標達成の節目 |

**刺さる理由**
「がんばった自分への記念」「トレーナーへの感謝を形にする」という価値が明確。
特にダイエット・産後ボディメイク・大会出場前後などの感情的な節目に刺さりやすい。

---

#### C. 会員体験談SNS映像（ジムB側直接費用プラン）

| 項目 | 内容 |
|------|------|
| 内容 | ジムのInstagram・TikTok・HPで使える会員体験談ショート動画 |
| 納品尺 | 30秒〜1分（縦型） |
| 販売価格 | 30,000円（ジムが直接費用負担） |
| 制作側取り分 | 30,000円（100%） |
| 提供タイミング | コース卒業時に会員の許諾を取り撮影 |

**刺さる理由（ジム側の課題解決）**
パーソナルジムは新規集客が最大の課題。口コミ・体験談が最も効果的な集客手段。
体験談映像は広告より信頼性が高く、TikTok・Reels活用で拡散効果が大きい。
内製する時間もスキルもないジムが多く、外注すると高い。FilmLinkなら安価に解決できる。

## 業界別 想定される懸念点と対処法

### 介護施設
| 懸念点 | 対処法 |
|--------|--------|
| 入居者・家族の同意取得が大変 | 同意書テンプレートをFilmLink側で用意し、施設側の負担を最小化 |
| 他の入居者が映り込む | 撮影エリア・時間帯を事前調整。個室・専用スペースを優先使用 |
| 認知症など撮影が難しい入居者がいる | 撮影可能な方を対象に限定。施設スタッフと事前に確認 |
| 施設が「追加業務になる」と嫌がる | 施設側の仕事は「案内と日程調整のみ」と明確に伝える |

### 工務店
| 懸念点 | 対処法 |
|--------|--------|
| 施主が引き渡し当日に映像を意識できるか | 見積もり段階でオプション提示。引き渡し前に再度確認する |
| 天候・工事遅延で撮影日が変わる | 日程調整は柔軟に対応。撮影は引き渡し日±1週間以内でOKとする |
| 施主がSNSシェアを嫌がる | SNSシェアはオプション。映像はあくまで施主個人のものとして提供 |

### パーソナルジム
| 懸念点 | 対処法 |
|--------|--------|
| 体型を映像に残すことへの抵抗感 | ビフォーアフター映像は任意。顔出し不要プランも用意する |
| SNSシェアを嫌がる会員 | SNS使用は別途許諾制。映像は個人向けとSNS用で分けて設計 |
| ジムが忙しくて案内する余裕がない | 申し込みページのQRコードを渡すだけで完結する仕組みを作る |
