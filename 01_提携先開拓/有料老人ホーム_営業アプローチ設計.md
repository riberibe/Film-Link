## 1. 優先施設の絞り込み基準

### 優先度 S（即アプローチ）
- 月額35万円以上
- 都心エリア（渋谷・文京・世田谷・大阪市天王寺/阿倍野）
- 理由: 入居者の購買力が最高水準。プランC（9万円）が刺さりやすい。

### 優先度 A（第二波）
- 月額25〜34万円
- 首都圏・大阪市内
- 理由: 購買力十分。プランA（3.5万円）の訴求が軸。

### 優先度 B（第三波）
- 月額20〜24万円
- 千葉・周辺エリア
- 理由: 母数は多いが成約単価が下がる。実績が出てから攻める。

## 3. 初回お問い合わせ文面草案

> **⚠️ リベ様確認待ち。GOサインをもらってから送付すること。**
> フォーム自動送信ツール（`ツール/form_sender/`）で一括送信する想定。

### 送付対象
施設の「お問い合わせフォーム」経由 → 施設長・相談員・事務長に届く想定

### 件名
```
ご入居者様から家族へ「映像の贈り物」— 御施設へのご提案
```

### 本文
```
突然のご連絡失礼いたします。
映像制作サービス「FilmLink」を運営している〇〇と申します。

このたびは、御施設のご入居者様に向けた
「家族への映像メッセージサービス」のご提案でご連絡いたしました。

■ サービス概要
ご入居者様が、お子様・お孫様へ言葉と記憶を贈る短編映像を制作するサービスです。
プロのカメラマンが施設に伺い、インタビュー形式で撮影します。

■ 御施設へのメリット
・入居者様の生きがい創出・満足度向上につながるオプションとして提供できます
・販売価格の20%（約7,000〜18,000円）を御施設の収益としてお受け取りいただけます
・施設側の初期費用・運営コストは一切かかりません

■ 料金プラン（入居者様・ご家族のご負担）
・プランA（3〜5分・家族1〜2名へのメッセージ）: 35,000円
・プランC（10〜15分・家族全員への集大成ムービー）: 90,000円

まずは15分ほどお時間をいただき、詳細をご説明させていただけますでしょうか。
ご都合のよい日時をご返信いただければ幸いです。

何卒よろしくお願いいたします。

───────────────────
FilmLink
担当：〇〇
メール：〇〇
電話：〇〇
───────────────────
```

### レビューポイント（リベ様確認事項）
- [ ] 担当者名（本名 or 屋号で出すか）
- [ ] メールアドレス・電話番号を載せるか

## 4. 施設スタッフ向けトークスクリプト

> **⚠️ リベ様確認待ち。内容を確認の上、修正があればお知らせください。**

### 想定シーン
担当者（相談員・フロントスタッフ）がご入居者・ご家族に案内する場面

### 入居者への案内トーク
```
「○○様、最近ご家族への映像メッセージを残せるサービスが始まりました。
プロのカメラマンが来て、○○様のお話を映像にしてくれるんですよ。
お子様やお孫様へのメッセージを形に残せるんですが、ご興味ありますか？
ご家族は何名いらっしゃいますか？」

→ 家族1〜2名 → プランA（35,000円）を案内
→ 家族3名以上 → プランC（90,000円）を案内
```

### ご家族への案内トーク（面会時・電話時）
```
「○○様（入居者）がとてもお元気で、いつもご家族のお話をされています。
最近、入居者様から家族への映像メッセージを残せるサービスを導入しました。
プロカメラマンが来て、○○様のお話をお子様やお孫様への映像にまとめてくれます。
費用は35,000円〜で、ご入居者様かご家族様どちらでもお申込みいただけます。
ご関心があればご案内しますが、いかがでしょうか？」
```

### よくある質問への返し方
| 質問 | 返答 |
|---|---|
| どんな映像になるの？ | 「インタビュー形式で、今まで歩んできた人生や家族への想いを語っていただき、3〜15分の映像にまとめます」 |
| 施設でやっていいの？ | 「施設のご協力のもとで実施しています。ご安心ください」 |
| 高くない？ | 「プロが撮影・編集した映像で、DVDとデータでお渡しします。葬儀の遺影動画などと比べても手頃な価格です」 |
