## 1. 仕組みの全体像

### このツールが何をやっているか

人間が毎日手作業でやっていたことを、パソコンが代わりにやります。

```
【人間がやること（手作業の場合）】
①「有料老人ホームAA」のホームページを開く
②「お問い合わせ」ページに移動する
③ 名前・メール・電話番号・本文を入力する
④「送信」ボタンを押す
⑤ 次の施設でも①〜④を繰り返す × 何十件

【ツールがやること（自動化後）】
→ まったく同じことを、1件あたり約5〜10秒で自動実行
→ 1日50件なら約10〜15分で完了
→ パソコンが動いている間ずっと動き続ける
```

### ファイル構成（全部で5つのファイル）

```
C:\FilmLink\form_sender\
│
├── send_forms.js     ← ツール本体（触らなくてOK）
├── facilities.csv    ← 送信先リスト（業種ごとに差し替えて使う）
├── config.json       ← 設定ファイル（文面・件数はここを編集）
├── results.csv       ← 送信結果ログ（自動で生成される）
└── screenshots\      ← 送信前の画面キャプチャ（自動で生成される）
```

### 全体の流れ（日々の運用）

```
【毎日の流れ】

朝・決まった時間に
↓
PowerShell を開く
↓
コマンドを1行打つ（または自動実行）
↓
Chromeが自動で立ち上がり、施設サイトを次々と開いて送信
↓
上限件数に達したら自動停止（例: 50件送ったら止まる）
↓
results.csv で結果を確認
↓
週1回: スプレッドシートを見て返信があった施設にアポ設定
```

## 2. Windowsパソコンへの初期設定（初回のみ・約30分）

> **この作業は最初の1回だけです。一度やれば次回からは不要です。**

### ステップ1: Node.jsをインストールする

Node.jsとは、このツールを動かすための「エンジン」です。無料です。

1. Windowsパソコンで Chrome を開く
2. 「nodejs.org」と検索してアクセスする
3. 「LTS」と書いてある緑のボタンをクリックしてダウンロード
4. ダウンロードされたファイル（`.msi`ファイル）をダブルクリックして実行
5. 「Next」を押し続けて、最後に「Install」を押す
6. 完了したら「Finish」

**インストールの確認（できているか確かめる）:**
1. キーボードの `Windows キー` を押す
2. 「PowerShell」と入力して、出てきた「Windows PowerShell」をクリック
3. 黒い画面が開いたら、以下を入力して Enter
   ```
   node --version
   ```
4. `v20.xx.x` のような数字が表示されればOK

### ステップ2: ツールのフォルダをWindowsにコピーする

Macにある以下のフォルダを、USBメモリ・Google Drive・AirdropなどでWindowsにコピーします。

**Macのコピー元:**
```
/Users/ribetyan/Desktop/ファミリームービー/ツール/form_sender/
```

**Windowsのコピー先（このパスにフォルダを置く）:**
```
C:\FilmLink\form_sender\
```

> `C:\FilmLink\` フォルダが存在しない場合は、エクスプローラーで `C:\` を開いて「新しいフォルダ」で「FilmLink」というフォルダを作ってください。

### ステップ3: 必要なソフトを追加インストールする

1. PowerShell を開く（ステップ1と同じ方法）
2. 以下のコマンドを **1行ずつ** 入力して Enter

```powershell
cd C:\FilmLink\form_sender
```
（`form_sender` フォルダに移動する命令です）

```powershell
npm install
```
（必要なパーツを自動でダウンロードする命令です。1〜3分かかります）

```powershell
npx playwright install chromium
```
（自動操作用のブラウザをインストールする命令です。3〜5分かかります）

### ステップ4: 動作確認（ドライラン）

実際に送信する前に、正しく動くか確認します。

```powershell
node send_forms.js --dry
```

**画面の見方:**
- Chrome が自動で立ち上がる → 正常
- 施設のサイトが次々と表示される → 正常
- `ドライラン（送信なし）` と表示される → 正常（これはテストモードなので送信しません）
- 最後に `ドライラン完了: ○件` と出ればセットアップ完了

## 3. 毎日の操作手順

### 起動方法（毎日これだけ）

1. PowerShell を開く
2. 以下を入力して Enter

```powershell
cd C:\FilmLink\form_sender
node send_forms.js
```

**これだけです。** あとはパソコンが自動でやります。

### 実行中の画面の見方

```
==================================================
FilmLink フォーム送信ツール
モード: 全自動送信
未処理件数: 59件 / 本日の上限: 50件
==================================================

[1/59] アリア恵比寿南
  → 送信完了（入力: name, email, phone, subject, body）
[2/59] アリア松濤
  → 送信完了（入力: name, email, phone, body）
[3/59] チャームプレミア グラン 松濤
  → スキップ（URL未設定）
...
[50/59] ○○施設
  → 送信完了

本日の送信上限（50件）に達しました。残り9件は明日以降に送信されます。
```

- `送信完了` → フォームへの入力・送信が成功
- `スキップ（URL未設定）` → フォームURLが未登録のためスキップ（問題なし）
- `エラー` → そのサイト側の問題（次回以降に自動リトライ）

### 正常に止めたいとき（途中停止）

PowerShellの画面で **`Ctrl + C`** を押す。  
→ 途中まで送信した分は results.csv に保存されているので、データは失われません。

### 緊急停止（パソコンの電源を切る）

そのまま電源を切っても大丈夫です。送信済みの施設のステータスは都度 facilities.csv に書き込まれているため、次回起動時は続きから再開します。

## 4. 文面を変えたいとき

文面は `config.json` というファイルに書いてあります。メモ帳で開いて編集できます。

### 開き方

1. エクスプローラーで `C:\FilmLink\form_sender\` を開く
2. `config.json` を右クリック → 「メモ帳で開く」

### 編集する場所

```json
"message": {
  "subject": "← ここがメールの件名",
  "body": "← ここが本文（\nは改行を意味する）"
}
```

**現在の件名:**
```
ご入居者様からご家族への「映像メッセージ」オプション — FilmLinkよりご提案
```

**現在の本文（要約）:**
```
突然のご連絡失礼いたします。FilmLinkの小林です。
ギフトムービーオプションのご提案です。
提供オプション型の仕組みで、施設様の初期費用ゼロです。
まずは10〜15分お時間をいただけますか。
```

### 注意事項

- 文中の改行は `\n` と書く（Enterキーではなく、`\n` という2文字を入力する）
- ダブルクォーテーション `"` を文中に使いたい場合は `\"` と書く
- 保存は `Ctrl + S`
- 業種ごとに文面を変えたい場合は、業種別のフォルダを作って config.json をそれぞれ置く（後述）

## 5. 1日の送信件数を変えたいとき

`config.json` の以下の数字を変更します。

```json
"settings": {
  "daily_limit": 50,   ← この数字を変える
  ...
}
```

**目安:**
| 件数 | 所要時間 | 推奨シーン |
|---|---|---|
| 20件 | 約3〜5分 | テスト・様子見のとき |
| 50件 | 約8〜12分 | 通常運用（デフォルト） |
| 100件 | 約15〜25分 | リスト消化を急ぐとき |
| 300件 | 約1時間 | フル稼働時 |

> 300件以上を毎日送ると、一部サイトでロボット検知（CAPTCHA）が発動する場合があります。その場合はエラーとして記録され、スキップされます。

## 6. 新しい業種を追加したいとき

新しい業種（例: 結婚式場）を追加するには、以下の3ステップです。

### ステップ1: スプレッドシートに新しいシートを追加する

1. 営業管理CRMを開く  
   → https://docs.google.com/spreadsheets/d/19vyVMkO2BsgNLH_ECjwYZ4esbS8Bq63nYYFWhf3oxX4
2. 画面下の `+` ボタンをクリックして新しいシートを追加
3. シート名を業種名にする（例: `営業管理_結婚式場`）
4. 以下の列を1行目に入力する（既存シートからコピーすると楽）

```
No. | 施設名 | 都道府県 | 市区町村 | HP_URL | フォームURL | ステータス
```

5. 企業情報を1行ずつ入力していく（Jarvisが代わりに収集します）

### ステップ2: CSVとしてダウンロードする

1. 追加したシートを開いた状態で
2. メニューの「ファイル」→「ダウンロード」→「カンマ区切り形式（.csv）」
3. ダウンロードされたCSVファイルを確認する

### ステップ3: フォルダに配置して実行する

1. ダウンロードしたCSVを `C:\FilmLink\form_sender\` にコピー
2. `facilities.csv` という名前に変更（既存の facilities.csv はバックアップとして `facilities_老人ホーム.csv` などに改名しておく）
3. `config.json` の文面を業種に合わせて変更する
4. `node send_forms.js` を実行

> **業種ごとの文面切り替えについて**  
> 将来的に複数業種を同時並行で運用する場合は、業種ごとにフォルダを分けるか、  
> `--csv 結婚式場.csv` のようにコマンドで指定できるよう改修予定。

## 7. 結果の確認方法

### 送信ログを見る（results.csv）

1. エクスプローラーで `C:\FilmLink\form_sender\` を開く
2. `results.csv` をダブルクリック → Excelで開く
3. 「送信結果」列を確認

| 送信結果 | 意味 |
|---|---|
| 送信完了 | 正常に送信できた |
| スキップ（URL未設定） | フォームURLが空のためスキップ |
| 送信ボタン未発見 | フォームはあったが送信ボタンが見つからなかった |
| エラー | サイトが開けなかった・タイムアウトなど |

### 送信前の画面を確認する（screenshots フォルダ）

`C:\FilmLink\form_sender\screenshots\` フォルダを開くと、送信直前の画面キャプチャが全件分保存されています。「本当にフォームに入力できているか」の証拠として確認できます。

### スプレッドシート（CRM）の確認

送信済みの施設は `facilities.csv` のステータスが `送信済み` に自動更新されます。  
週1回程度、このデータをスプレッドシートに反映してください（Jarvisが代行可能）。

## 8. よくあるトラブルと対処法

### Q: 「node は内部コマンドとして認識されません」と出る

**→ Node.js がインストールされていません。** ステップ1からやり直してください。

### Q: Chrome が途中で固まる・タイムアウトになる

**→ そのサイトの問題です。** ツールはエラーを記録して次の施設に進みます。  
エラー件数が多い場合はJarvisに報告してください。

### Q: 「CAPTCHA」と書いた画面が出て止まる

**→ そのサイトでロボット検知が発動しました。** エラーとして記録されスキップされます。  
この施設は手動で対応するか、しばらく間隔を空けてから再送します。

### Q: 同じ施設に2回送ってしまった

**→ facilities.csv のステータスが「未処理」に戻っていないか確認してください。**  
送信済みは「送信済み」になっているはずです。誤って「未処理」に戻してしまうと再送されます。

### Q: ツールを止めてしまったが、続きから再開できるか

**→ できます。** `node send_forms.js` をもう一度実行するだけで、  
送信済み（ステータス: 送信済み）の施設は自動スキップされ、続きから再開されます。

## 9. 自動実行スケジュールの設定（上級）

毎日決まった時間に自動でツールが起動するよう設定できます。

### Windowsタスクスケジューラーを使う

1. キーボードの `Windows キー` を押す
2. 「タスクスケジューラ」と検索して開く
3. 右側の「タスクの作成」をクリック
4. 以下のように設定する:

**全般タブ:**
- 名前: `FilmLink 毎日送信`
- 「最上位の特権で実行する」にチェック

**トリガータブ:**
- 「新規」→「毎日」→ 時刻を指定（例: 毎朝9:00）

**操作タブ:**
- 「新規」→ 操作: 「プログラムの開始」
- プログラム: `C:\Program Files\nodejs\node.exe`
- 引数: `send_forms.js`
- 開始（作業フォルダ）: `C:\FilmLink\form_sender`

5. 「OK」で保存

これで毎朝9:00に自動で起動し、50件送信したら自動で止まります。
