## 事業概要

**事業名**: FilmLink  
**本質**: B2B2C型。既存事業者に映像制作をオプションとして組み込んでもらい、レベニューシェアで収益を得る。提携先が営業してくれるので自社営業コストがかからない構造。  
**現在フェーズ**: Phase 0B（有料老人ホーム・初回コンタクト直前）— **本番送信リベ様確認待ち**


### 実行コマンド

```bash

# ドライラン（フォームを開いて入力まで。送信ボタンは押さない）

# 本番送信（daily_limitで設定した件数だけ送信して自動停止）

```
